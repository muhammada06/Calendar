FAQs

Q1: Can I use the calendar without an internet connection?

A: No, you cannot.



Q2: How do I download my calendar events?

A: Login → Download Event → Enter starting and ending date and time → Download Events → You will now get a text file with all the events from that time period.



Q3: How do I add a calendar event?

A: Login → Click the the "+" button at the bottom right of the screen → Enter title, starting date, ending date, and description → Click "Add Event"



Q4: How do I delete a calendar event?

A: Login → Hover over the event you want to delete → Click "Delete Event" → Select "Yes, delete this event" → Click "Submit" and you'll be redirected to the homepage



Q5: How do I change an event’s time or date?

A: Hover over the event → Click "Delete" → Add event with the current details.
